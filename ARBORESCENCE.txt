================================================================================
                              STATISTIQUES
================================================================================

FRONTEND (React + TypeScript + Vite):
  • 16 Pages
  • 3 Composants réutilisables
  • 2 Services (API, Auth)
  • 1 Store Zustand
  • Dependencies: Axios, Lucide Icons, Tailwind CSS, Framer Motion, Recharts

BACKEND (Spring Boot + Maven):
  • 10 REST Controllers
  • 11 Entités JPA
  • 9 Repositories (Spring Data)
  • 9 Services métier
  • 3 Classes de sécurité JWT
  • Support GraphQL
  • BD: Application properties configurée

OUTILS & FRAMEWORKS:
  Frontend: React 18, TypeScript, Vite, Tailwind CSS, ESLint
  Backend: Spring Boot, Maven, JPA/Hibernate, JWT, GraphQL

================================================================================
                              DESCRIPTION
================================================================================

FONCTIONNALITÉS PRINCIPALES:

Admin:
  - Dashboard avec statistiques
  - Gestion des utilisateurs
  - Gestion des rapports
  - Support & messagerie
  
Encadrant (Supervisor):
  - Dashboard avec étudiants assignés
  - Gestion des deadlines
  - Visualisation des rapports étudiants
  
Étudiant:
  - Dashboard personnel
  - Voir les deadlines de son encadrant
  - Soumettre des rapports
  - Demandes d'encadrement
  - Notifications
  
Utilisateur:
  - Profil personnel
  - Événements personnels
  - Messages
  - Paramètres
